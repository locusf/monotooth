Monotooth

Monotooth is a library framework that enables Mono & .NET developers to use bluetooth in their projects. Current release includes partial support for bluez, so the current situation only supports linux. Windows version will be released at some point in the future (for information on how it could be done, see Bluecove svn repository).

Features:

  • RFCOMM connections
  • Device discovery
  • Service discovery
  • Service connections (since 0.0.2 release, 0.0.3 release has better code for service connection)
  • Good design, the library should be easy to use
  • OS-independent connections and devices, meaning that connections are created by the value of the operating system.
